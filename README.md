Tegra是NVIDA推出的基于ARM架构通用处理器品牌（即CPU，NVIDIA称为“Computer on a chip”片上计算机），能够为便携设备提供高性能、低功耗体验。   
Tegra是一种采用单片机系统设计（system-on-a-chip）芯片，它集成了ARM架构处理器和NVIDIA的GeforceGPU，并内置了其它功能，产品主要面向小型设备。
